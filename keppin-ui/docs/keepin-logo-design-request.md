# keepin 로고 디자인 요청 브리프

- 문서 버전: v1.0
- 작성일: 2026-02-23
- 목적: 외부 디자이너/에이전시/AI 툴에 전달할 로고 제작 요청서

## 1) 프로젝트 요약
`keepin`은 소중한 인연을 놓치지 않도록 돕는 관계 관리 앱입니다.  
핵심 가치는 `기억 -> 관리 -> 행동`이며, 로고는 "따뜻하지만 명확한 실행 중심" 이미지를 전달해야 합니다.

## 2) 브랜드 키워드
- 따뜻함
- 명료함
- 절제됨
- 실용성
- 신뢰감

## 3) 로고 콘셉트 방향 (필수)
### 기본 콘셉트
- `Connection K`를 기반으로 설계
- `K` 조형 안에 "나-타인-관계 연결"의 의미가 드러나야 함

### 의미 매핑
- Stem(기둥): 나/중심축
- Curved arms(곡선): 관계 연결선
- Dots(점): 연결된 사람들

## 4) 스타일 가이드 (필수)
- 과도한 장식/복잡한 디테일 금지
- 앱 아이콘(작은 사이즈)에서도 식별 가능한 단순 형태
- 라이트/다크 배경 모두에서 대비가 유지되어야 함
- 텍스트 워드마크(`keepin`)와 심볼 조합형 + 심볼 단독형 모두 제안
- iOS/Android 앱 아이콘 적용 가능한 정사각형 구조 고려

## 5) 컬러 가이드 (TDS 규칙 준수)
### 고정 조건
- 라이트 모드 배경: `#FFFFFF`
- 다크 모드 배경: `#1A1E27`

### 사용 가능한 브랜드 팔레트
- Main: `#1A1E27`
- Positive: `#21C389`
- Negative: `#FF4D4D`
- Sub Green: `#34C185`
- Sub Blue: `#3F80FF`
- Sub Pink: `#FE88E4`
- Neutral Gray Scale: `#F4F6F8`, `#E8EBF0`, `#D8DCE2`, `#C4CAD4`, `#A5ADBD`, `#8E95A3`, `#595F72`, `#353C49`, `#1A1E27`

### 컬러 사용 비율
- Neutral 70 / Main 20 / Accent 10 원칙 유지
- 포인트 컬러는 강조 포인트에만 제한적으로 사용

## 6) 금지사항 (중요)
- 그라디언트 남용, 3D 효과, 과한 그림자 금지
- "최종본 느낌"의 트렌디 장식보다 장기 운영 가능한 형태 우선
- 가독성 떨어지는 얇은 획, 작은 사이즈에서 깨지는 형태 금지
- 유사 서비스 로고와 혼동 가능한 구조 금지

## 7) 납품 산출물 (필수)
1. 심볼 단독형: SVG, PNG(투명 배경)
2. 조합형(심볼 + keepin 워드마크): SVG, PNG
3. 단색 버전: Black/White 각각
4. 다크 배경용/라이트 배경용 컬러 버전
5. 앱 아이콘 목업 (1024x1024 기준)
6. 최소 사이즈 가이드(예: 16px, 24px, 32px, 48px)에서 식별성 검증 이미지
7. 클리어 스페이스/오용 금지 예시 1장

## 8) 결과물 평가 기준
- 의미 전달: "연결"과 "관리"가 직관적으로 읽히는가
- 식별성: 작은 크기에서도 `K` 인지가 가능한가
- 확장성: 앱 아이콘, 스플래시, 헤더, 문서에 일관 적용 가능한가
- 시스템 적합성: TDS 컬러/명도/대비 규칙에 맞는가

## 9) 디자이너에게 바로 전달할 요청문 (복붙용)
```text
keepin 앱 로고 디자인 요청드립니다.

브랜드 핵심은 "소중한 인연을 기억하고, 관리하고, 행동으로 연결한다"입니다.
로고는 Connection K 콘셉트로, K 형태 안에 관계 연결의 의미가 보이게 해주세요.

[필수 조건]
- 심볼 단독형 + 워드마크 조합형 제안
- 작은 사이즈(앱 아이콘)에서도 식별 가능해야 함
- 라이트(#FFFFFF)/다크(#1A1E27) 배경 모두 대응
- 과한 장식, 3D, 복잡한 디테일은 지양
- TDS 컬러 규칙 준수

[컬러]
- Main #1A1E27
- Positive #21C389
- Negative #FF4D4D
- Sub #34C185, #3F80FF, #FE88E4
- Neutral Gray 스케일 활용
- 비율은 Neutral 70 / Main 20 / Accent 10

[납품]
- SVG + PNG(투명) / 심볼, 조합형, 단색 버전 포함
- 앱 아이콘 1024px 버전
- 최소 사이즈 식별성 검증 이미지
- 클리어 스페이스/오용 가이드 1장
```
